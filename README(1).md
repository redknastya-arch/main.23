# Guess the Number & AI Assistant

## Description
A Python project that combines:
- AI assistant via OpenRouter API
- Console game "Guess the Number"

## Requirements
- Python 3.11+
- requests

## Installation

```bash
pip install requests
```

## Run

```bash
python main.py
```

## Features
- Chat with AI
- Number guessing game
- Input validation
- Simple console interface

## Exit
Type:

```text
exit
```

to close the program.
